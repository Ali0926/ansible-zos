# Ansible for z/OS 

This repo contains samples and starter code for training in how to use Ansible with the ibm_zos_core collection. It also contains descriptions of the lab exercises.

Fork the repo and do your work in you own copy of the repo. 

## Expectations for starting the course 

#### Your Ansible background 

- You have Ansible installed on an system you can use as the Control Node.
- You have a basic working knowledge of Ansible, equivalent to an introductory training course or a couple of weeks' experience.

#### Your z/OS background 

- You have a 3270 emulator installed on your local system.
- Z what, now?

Labs are documented here: [Labs](labs/labs.md)
